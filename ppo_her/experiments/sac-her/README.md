# SAC-HER-2

Test the peformance of SAC and SAC-HER in order to compare with PPO and PPO-HER.  This uses the optimized sac-her parameters (we only optimized for static).  These results were used [PPO-HER](https://arxiv.org/abs/2410.22524) Figures 1 and 2 - just the SAC and SAC-HER conditions.  For PPO and PPO-HER conditions, see the ppo-her experiment.
